## Business Problem

Organizations generate large volumes of sales data but often struggle to derive actionable insights.

This dashboard was developed to analyze sales performance, profitability, customer behavior, and regional trends to support data-driven decision-making.

- DAX
- Microsoft Excel
- Data Visualization
